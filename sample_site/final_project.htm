Model of the Public Internet

by: Rani Fields, Peter Grabowski, Aditya Nemani, Jason Hunsberger

Goals:

Our goal was to use an adjacency matrix to visualize the public Internet and encode information that would be useful to a networking or information security professional. To accomplish this, we encoded the entire Internet Protocol v4 (IPv4) network address space using a Hilbert curve to keep related networks adjacent to each other.

We believe this visualization is a useful proof of concept for how to present, navigate, and enable exploration of a massive dataset related to the entire public Internet. With this visualization users will be able to answer questions such as: which IP blocks are explicitly reserved for special use addresses? which IP blocks are owned by particular countries? what network blocks are the used most prevalently? and many others.
